# 页面原型说明

这是 `JCC Ticket Cloudflare Lite` 的可点击静态页面原型。

打开方式：

```text
直接用浏览器打开 index.html
```

当前支持体验：

- 电脑端左侧导航，带图标并固定显示
- 平板端顶部横向导航，带图标
- 手机端底部导航，带图标
- 登录页
- 首页汇总
- 快速录入
- 存票人列表
- 存取记录列表
- 个人历史
- 快速录入页下方的计算工具，默认折叠，点击后展开
- 导入导出
- 系统信息与维护页
- 模拟新增存入/取用记录
- 模拟余额不足拦截
- 模拟作废/恢复记录
- 计算工具显示计算链，支持逐步核对和删除单步
- 计算工具支持本次数量加减和快捷数量选择
- 计算链中的每一步支持修改加减方向、价格和数量
- 计算工具支持常用价格加减、自定义价格加减
- 计算工具支持原始、抹零、取整三种结果分别回填到快速录入
- 存票人列表分页和顺序编号
- 存取记录分页和顺序编号
- 个人历史分页和顺序编号
- 个人历史支持存票人搜索选择
- 个人历史桌面端使用表格展示，手机端自动转为卡片式布局
- 个人历史显示序号、时间、类型、票数、操作后余额、状态和备注
- 记录时间精确到分钟
- 分页支持首页、上一页、页码跳转、每页条数、下一页、末页
- 记录筛选支持日期范围、查询、重置和结果摘要
- 作废/恢复记录确认弹窗
- 电脑端顶部栏和侧边栏固定
- 快速录入支持存票人搜索选择
- 快速录入支持取用后/存入后余额预览
- 首页统计卡和最近记录可点击进入筛选后的记录列表
- 手机端更多页包含个人历史、导入导出、系统信息和退出登录

当前不包含：

- 登录真实校验
- Cloudflare D1 数据连接
- 后端 API
- 数据持久化

这个原型用于先验证页面布局和操作流，确认后再进入正式前端和 Worker API 实现。
