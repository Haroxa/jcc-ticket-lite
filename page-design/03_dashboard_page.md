# 03 首页汇总

## 1. 页面目标

快速看到当前存票总览，并进入常用操作。

## 2. 桌面端布局

```text
┌────────────────────────────────────────────────────┐
│ 首页汇总                         [新增记录] [导出] │
├────────────┬────────────┬────────────┬────────────┤
│ 当前总存票 │ 存票人数   │ 今日存入   │ 今日取用   │
├─────────────────────────┬──────────────────────────┤
│ 存票人余额排行           │ 最近存取记录              │
│                         │                          │
│ 1. 张三      1200        │ 张三 存入 100             │
│ 2. 李四       800        │ 李四 取用 50              │
│ 3. 王五       300        │                          │
└─────────────────────────┴──────────────────────────┘
```

## 3. 核心信息

统计块：

- 当前总存票
- 存票人数
- 今日存入
- 今日取用

列表：

- 余额排行
- 最近记录

## 4. 操作

- 点击 `新增记录` 进入快速录入。
- 点击余额排行中的存票人，进入个人历史。
- 点击最近记录中的记录，进入记录详情或定位到记录列表。

## 5. 空状态

如果没有数据：

- 显示 `暂无存票记录`
- 主按钮：`新增第一条记录`
- 次按钮：`导入 Excel 数据`
