# 07 个人历史页

## 1. 页面目标

查看某个存票人的余额、合计和完整流水。

## 2. 布局

```text
┌────────────────────────────────────────────────────┐
│ 个人历史                                           │
│ 存票人：[搜索/选择张三]              [导出此人记录] │
├────────────┬────────────┬────────────┬────────────┤
│ 当前余额   │ 存入合计   │ 取用合计   │ 有效记录数 │
├────────────────────────────────────────────────────┤
│ 日期范围：[____ 至 ____] 类型：[全部] 状态：[正常] │
├────────────────────────────────────────────────────┤
│ 时间线/表格                                         │
│ 06-24 存入 100  余额变化 +100                       │
│ 06-23 取用 50   余额变化 -50                        │
└────────────────────────────────────────────────────┘
```

## 3. 汇总信息

- 当前余额
- 存入合计
- 取用合计
- 有效记录数

## 4. 交互

- 搜索并切换存票人。
- 日期筛选。
- 类型筛选。
- 导出当前存票人的记录。

## 5. 空状态

如果存票人没有记录：

- 显示 `该存票人暂无记录`
- 提供 `新增记录` 按钮
