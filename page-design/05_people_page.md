# 05 存票人管理页

## 1. 页面目标

维护存票人基础信息，并快速查看余额。

## 2. 布局

```text
┌────────────────────────────────────────────────────┐
│ 存票人管理                 [新增存票人]            │
├────────────────────────────────────────────────────┤
│ 搜索：[________________] 状态：[全部 v]            │
├────────────────────────────────────────────────────┤
│ 存票人       当前余额      状态      最近更新      操作 │
│ 张三         1200          正常      06-24         历史 │
│ 李四          800          正常      06-23         历史 │
│ 王五            0          停用      06-20         历史 │
└────────────────────────────────────────────────────┘
```

列表要求：

- 第一列显示顺序编号。
- 默认每页显示 10 到 20 条，原型阶段可用 5 条方便验证。
- 底部显示上一页、下一页、当前页和总数。

## 3. 操作

- 新增存票人
- 编辑存票人
- 停用/启用
- 查看历史

## 4. 新增/编辑弹窗

字段：

- 名称
- 别名
- 状态
- 备注

规则：

- 名称必填。
- 名称不建议重复。
- 停用后不出现在快速录入默认列表。
