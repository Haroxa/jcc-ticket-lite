# 01 应用外壳与导航

## 1. 电脑端布局

适用宽度：

```text
>= 1100px
```

布局：

```text
┌────────────────────────────────────────────────────────────┐
│ 顶部栏：JCC 存票管理       当前账号：管理员      退出登录 │
├───────────────┬────────────────────────────────────────────┤
│ 侧边导航       │ 主内容区                                   │
│               │                                            │
│ 首页汇总       │ 页面标题 + 当前页面操作                     │
│ 快速录入       │ 页面主体，可使用双栏或三栏                  │
│ 存票人         │                                            │
│ 存取记录       │                                            │
│ 个人历史       │                                            │
│ 导入导出       │                                            │
│ 设置           │                                            │
└───────────────┴────────────────────────────────────────────┘
```

特点：

- 左侧固定导航。
- 表格优先显示完整列。
- 首页和快速录入可以使用双栏，计算工具作为快速录入内的辅助模块。
- 适合键盘和鼠标快速操作。

## 2. 平板端布局

适用宽度：

```text
768px - 1099px
```

布局：

```text
┌────────────────────────────────────────────────────────────┐
│ 顶部栏：JCC 存票管理                  当前账号 / 退出登录 │
├────────────────────────────────────────────────────────────┤
│ 横向导航：首页 录入 存票人 记录 历史 更多                   │
├────────────────────────────────────────────────────────────┤
│ 主内容区                                                    │
│ 页面标题 + 操作区                                           │
│ 内容使用 1 到 2 栏，根据空间自动折行                       │
└────────────────────────────────────────────────────────────┘
```

特点：

- 不使用固定左侧栏，避免横向空间被占用。
- 使用顶部横向导航。
- 首页统计卡片使用 2 到 4 列。
- 表格保留主要列，次要信息收进卡片详情或隐藏。

## 3. 手机端布局

适用宽度：

```text
< 768px
```

布局：

```text
┌──────────────────────────────┐
│ 顶部栏：JCC 存票管理 / 用户  │
├──────────────────────────────┤
│ 当前页面内容                  │
│ 单列布局                      │
│ 表格转卡片                    │
├──────────────────────────────┤
│ 底部导航：首页 录入 记录 ... │
└──────────────────────────────┘
```

底部导航：

```text
首页 | 录入 | 记录 | 存票人 | 更多
```

特点：

- 底部导航固定显示最高频入口。
- 表格改成卡片列表。
- 快速录入单列展示。
- 按钮和输入框高度不低于 44px。

## 4. 顶部栏内容

左侧：

- 系统名称：JCC 存票管理

右侧：

- 当前用户显示名
- 角色标签
- 退出登录按钮

手机端可以隐藏角色标签，只保留用户名和退出入口。

## 5. 导航顺序

完整页面顺序：

1. 首页汇总
2. 快速录入
3. 存票人
4. 存取记录
5. 个人历史
6. 导入导出
7. 系统信息

计算工具不作为主导航项，放在快速录入页下方。

多端导航映射：

| 页面 | 电脑端 | 平板端 | 手机端 |
|---|---|---|---|
| 首页汇总 | 左侧导航 | 顶部导航 | 底部：首页 |
| 快速录入 | 左侧导航 | 顶部导航 | 底部：录入 |
| 存取记录 | 左侧导航 | 顶部导航 | 底部：记录 |
| 存票人 | 左侧导航 | 顶部导航 | 底部：存票人 |
| 个人历史 | 左侧导航 | 顶部导航 | 更多 |
| 导入导出 | 左侧导航 | 更多 | 更多 |
| 系统信息 | 左侧导航 | 更多 | 更多 |

## 6. 全局状态

需要统一处理：

- 加载中
- 保存中
- 保存成功
- 保存失败
- 登录过期
- 无权限

提示方式：

- 页面顶部轻提示
- 表单字段旁错误
- 关键危险操作使用确认弹窗
