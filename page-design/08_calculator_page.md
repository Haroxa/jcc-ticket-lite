# 08 计算工具模块

## 1. 页面目标

复刻 Excel v9 的常用价格计算器，帮助快速算出票数并应用到录入页。

该模块建议嵌入 `快速录入` 页下方，不作为第一版独立主页面。

## 2. 布局

```text
┌──────────────────────────────────────────────┐
│ 快速录入页下方：计算工具                      │
├──────────────────────┬───────────────────────┤
│ 本次数量 [ 1 ]        │ 计算结果               │
│ 自定义价格 [____]     │ 原始总数：1298          │
│                      │ 向下抹零：1200          │
│ 常用价格              │ 向上取整：1300          │
│ [+99] [-99]           │ 当前选择：1200          │
│ [+199] [-199]         │                       │
│ [+299] [-299]         │ [应用到快速录入]        │
│ [+366] [-366]         │ [清空]                 │
└──────────────────────┴───────────────────────┘
```

## 3. 常用价格

默认：

```text
99、199、299、366、520、999、1314、3000
```

## 4. 交互

- 点击 `+价格` 增加。
- 点击 `-价格` 减少。
- 可输入数量，表示一次加减多个。
- 可输入自定义价格。
- 可选择原始总数、向下抹零、向上取整。
- 点击应用后跳转快速录入，并把票数填入表单。
  - 如果已经在快速录入页，则直接回填票数字段。

## 5. 计算规则

```text
原始总数 = 所有加减明细合计
向下抹零 = FLOOR(原始总数, 100)
向上取整 = CEILING(原始总数, 100)
```
