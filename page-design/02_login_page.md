# 02 登录页

## 1. 页面目标

让管理员、录入员、查看者登录系统。

## 2. 布局

```text
┌──────────────────────────────┐
│ JCC 存票管理                  │
│ 云端存票记录与余额汇总        │
│                              │
│ 用户名                        │
│ [____________________]       │
│                              │
│ 密码                          │
│ [____________________]       │
│                              │
│ [ 登录 ]                      │
│                              │
│ 登录失败提示                  │
└──────────────────────────────┘
```

## 3. 字段

- 用户名
- 密码

## 4. 交互

- 回车提交。
- 登录中按钮显示 `登录中...`。
- 登录失败显示中文错误。
- 登录成功进入首页汇总。

## 5. 状态

- 初始状态
- 输入中
- 登录中
- 登录失败
- 登录成功
