# 09 导入导出页

## 1. 页面目标

迁移 Excel v9 数据，并提供日常备份能力。

## 2. 布局

```text
┌────────────────────────────────────────────────────┐
│ 导入导出                                           │
├──────────────────────────┬─────────────────────────┤
│ 导入 Excel v9 CSV         │ 导出数据                 │
│ [选择 CSV 文件]           │ [导出全部流水]            │
│ [预览导入]                │ [导出余额表]              │
│                          │ [导出当前筛选结果]        │
├────────────────────────────────────────────────────┤
│ 导入预览                                             │
│ 总行数 / 可导入 / 错误 / 警告                         │
│ 错误列表                                             │
│ [确认导入]                                           │
└────────────────────────────────────────────────────┘
```

## 3. 导入流程

1. 选择 CSV。
2. 预览导入。
3. 显示错误和警告。
4. 管理员确认。
5. 写入数据。
6. 重新计算余额。
7. 写导入日志。

## 4. 导出类型

- 全部流水
- 当前筛选流水
- 存票人余额
- 操作日志

## 5. 错误显示

错误列表字段：

- 行号
- 存票人
- 原始类型
- 原始票数
- 错误原因
