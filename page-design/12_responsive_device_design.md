# 12 多端适配设计

## 1. 目标

项目需要支持电脑、手机、平板等多设备观看、录入和调整数据。

设计目标：

- 同一套页面适配不同屏幕。
- 高频操作在手机上也能完成。
- 表格类信息在小屏上改为卡片。
- 平板不强行使用电脑侧边栏。
- 数据操作按钮在多端位置稳定。

## 2. 断点设计

```text
手机：< 768px
平板：768px - 1099px
电脑：>= 1100px
```

## 3. 导航策略

### 电脑

- 左侧固定导航。
- 顶部显示系统名、账号和退出。
- 主内容区适合表格、双栏和数据密度更高的页面。

### 平板

- 顶部横向导航。
- 主要页面直接显示。
- 低频页面放入 `更多`。
- 内容以两栏为主，不够时自动变一栏。

### 手机

- 底部导航。
- 只保留最高频入口：

```text
首页 | 录入 | 记录 | 存票人 | 更多
```

- 个人历史、导入导出、系统信息放入更多。
- 计算工具放在快速录入页下方。

## 4. 页面适配规则

| 页面 | 电脑 | 平板 | 手机 |
|---|---|---|---|
| 首页汇总 | 统计四列，排行和最近记录双栏 | 统计二到四列，内容双栏 | 统计两列，列表上下排列 |
| 快速录入 | 表单和存票人信息双栏 | 表单上方，最近记录下方或右侧 | 单列表单，最近记录在下方 |
| 存票人 | 表格 | 简化表格 | 卡片列表 |
| 存取记录 | 完整表格 | 主列 + 展开详情 | 卡片列表 |
| 个人历史 | 汇总卡 + 表格 | 汇总卡 + 简化表格 | 汇总卡 + 时间线卡片 |
| 计算工具模块 | 快速录入页下方双栏 | 快速录入页下方双栏或单栏 | 快速录入页下方单列，大按钮 |
| 导入导出 | 双栏 | 双栏变单栏 | 单列，导入预览卡片 |
| 系统信息 | 版本信息和维护入口双栏 | 双栏变单栏 | 单列卡片 |

## 5. 表格转卡片规则

小屏不显示宽表格，统一转为卡片。

记录卡片结构：

```text
存票人        类型 + 票数
日期          状态
备注摘要
操作按钮
```

存票人卡片结构：

```text
存票人名称        当前余额
状态              最近更新
备注摘要
查看历史 / 编辑
```

## 6. 原型验证清单

每次实现页面后，至少检查：

- 1366px 宽电脑屏幕。
- 1024px 平板横屏。
- 768px 平板竖屏。
- 390px 手机竖屏。

检查项：

- 导航是否可见。
- 表单是否不用横向滚动。
- 表格是否转换为卡片。
- 保存、作废、恢复按钮是否容易点击。
- 统计数字是否不会挤在一起。
- 底部导航是否遮挡内容。
