# 06 存取记录页

## 1. 页面目标

查看、筛选、作废、恢复所有流水记录。

## 2. 布局

```text
┌────────────────────────────────────────────────────────────┐
│ 存取记录                              [导出当前结果]       │
├────────────────────────────────────────────────────────────┤
│ 存票人[____] 类型[全部] 状态[正常] 日期[____ 至 ____] 搜索 │
├────────────────────────────────────────────────────────────┤
│ 序号 日期   存票人  类型  票数  变化  状态  备注  操作       │
│ 1    06-24  张三    存入  100   +100  正常  ...   作废       │
│ 2    06-24  李四    取用   50    -50  正常  ...   作废       │
│ 3    06-23  王五    存入  200   +200  作废  ...   恢复       │
├────────────────────────────────────────────────────────────┤
│ 上一页                         第 1 / 10 页        下一页 │
└────────────────────────────────────────────────────────────┘
```

## 3. 筛选条件

- 存票人
- 类型
- 状态
- 日期范围
- 关键词

## 4. 分页与编号

记录列表必须支持分页。

规则：

- 第一列显示全局顺序编号，不是当前页内编号。
- 筛选后编号按筛选结果重新计算。
- 分页栏显示上一页、下一页、当前页、总页数、总记录数。
- 手机端仍保留编号，但以卡片字段显示。

## 5. 操作

### 作废

点击作废后弹窗：

```text
作废记录
请输入作废原因
[________________]
[取消] [确认作废]
```

### 恢复

恢复前二次确认。

如果恢复后余额会低于 0，后端拒绝恢复。

## 6. 行状态

- 正常记录：常规显示。
- 作废记录：降低对比度，显示作废原因。
- 存入：正数绿色。
- 取用：负数橙色。
